=================
shindanmaker.rb
=================

WHAT'S THIS
------------
mikutterから診断メーカーを自動ポストするプラグインです．

INSTALL
--------
mikutterのpluginディレクトリに放り込んでください．
最低必須バージョンは0.1.0.601では動いてます．

HOW TO USE
----------
このプラグインを入れた状態で診断メーカーのリンクを踏むと，自動的に診断結果が画面上部のテキストボックスに入ります．

設定画面の「診断メーカー」からタイムアウトが設定できます．
